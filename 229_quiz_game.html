Are You a Cyber Ops Expert?

A tv game show-like study game accessed before final assessments during Cyber and Information Operations courses. Created using CSS and javascript, it displays multiple choice questions followed by an increase or reduction in score. As the user's score increases or decreases, the gauge on the left animates up or down, successfully finishing the game when it reaches the top and losing if it reaches the bottom.

Features multiple difficulty levels, and one time use aids such as "Ask the expert", a 50/50 removing two of the incorrect options, and an ability to choose a second answer
